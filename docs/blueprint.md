# **App Name**: PharmaTrack

## Core Features:

- Drug Inventory: Maintain real-time tracking of drug quantities.
- Automated Distribution: Automatically deduct distributed medications from the inventory.
- Digital Record Keeping: Replace paper records with digital data linked via barcodes.
- Distribution Table: Store information with 'Barcode', 'Item Name', 'Quantity distributed', 'Service/Section', and 'Date' fields, the Barcode is a reference to the drug database.
- Automatic Name Retrieval: Use the Barcode reference in the distribution table to automatically get the medicine's name.
- Interactive Dashboard: An overview showing the most important stats.
- Stock Adjustment Tool: LLM powered tool to decide what kind of adjustments need to be done, for example, when expiration date is near, decide how much the stock should be reduced.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to evoke trust and tranquility, aligning with healthcare settings.
- Background color: Light, desaturated blue (#E8F5F9) to create a clean and calm backdrop.
- Accent color: Pale green (#C8E6C9) to highlight important actions and information.
- Body and headline font: 'PT Sans', a humanist sans-serif for a blend of modernity and warmth, suitable for both headings and body text.
- Use clear, professional icons related to medications and hospital departments.
- Design the layout to be intuitive, prioritizing ease of navigation and quick access to essential functions.
- Use subtle animations to provide feedback and enhance user experience.
